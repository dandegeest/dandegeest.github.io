SynestR

A SYNESthetic insTRument Exploration

Project Overview

A software synthesizer framework for SYNESthetic insTRument exploration and visualization in Processing

Technical Details

  • An extensible OOP framework for quick Synestheic Instrument experiments
  • Midi, Sequencing and Sound Synthesis using Java Sound API
  • Motion Graphics and Custom User Interfaces
  • Serial Communication with Arduino
  • View Code
  • Sound Synthesis
  • Midi
  • Motion Graphics
  • Sound Design
  • Java
  • Arduino
  • Processing

Project Goals

The project aims to create an immersive experience that explores the relationship between sound and visual elements, challenging traditional perceptions of sensory interaction.
