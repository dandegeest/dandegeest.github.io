Data Portrait

Interactive Data Visualization

Project Overview

An interactive data visualization project that transforms personal data into dynamic visual portraits. Using Processing, the project captures and processes various data points to create evolving, abstract representations of personal information. The visualization responds to user interaction, creating unique patterns and forms that reflect the underlying data structure while maintaining an aesthetic appeal.
