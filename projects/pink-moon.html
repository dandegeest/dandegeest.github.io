The Pink Moon

Interactive Installation & Live Performance Visuals

Project Overview

An interactive installation and projection mapping project created for a live musical performance, exploring the ethereal beauty of the Pink Moon through immersive visual elements and audience interaction. The setup featured two synchronized projectors casting layered visuals across the performance space, controlled via a custom Arduino interface that triggered visual transitions in response to both the music and performer inputs. The visuals combined pre-rendered animations with live graphic manipulation using Processing, allowing for real-time visual performance alongside the live music. This created a dynamic audiovisual experience where audience presence and sound influenced the visual atmosphere in subtle, poetic ways.
