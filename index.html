The Pink Moon

An interactive installation and projection mapping piece created for a live musical performance, exploring the ethereal beauty and symbolism of the lunar cycle. The visuals feature custom-designed LCD-style animations rendered through a simulated pixel display filter, evoking a nostalgic digital aesthetic. Through layered imagery, reactive lighting, and immersive projection, the work transforms the performance space into a dreamlike environment inspired by the transient glow of the full moon.

LogaRhythm

An interactive musical experience combining algorithmic composition with visual elements, exploring the intersection of mathematics and music. The installation features custom MIDI-generating code that drives both sound and visuals in real time, displayed on four reclaimed LCD TVs arranged in a simulated campfire formation. These salvaged screens emit flickering, abstract patterns and rhythmic visualizations, creating an intimate, meditative space where logic, nature, and sound converge.

SynestR

An interactive Processing sketch that merges generative sound and visual elements to create a unique, multisensory experience. It features a custom MIDI sequencing and playback system that dynamically composes and triggers audio in real time, synchronized with evolving graphics. The result is a responsive, algorithmically driven composition that explores the relationship between sound, code, and visual form.
